# Job Market Intelligence Pipeline

A 3-phase automated data application built to ingest, clean, and analyze live tech job market listings. 

## Project Overview
This project is built in evolutionary stages to demonstrate database architecture, python scripting, and modern workflow automation:
* **Phase 1 (Current):** Database Schema design in Supabase (PostgreSQL) and advanced SQL data cleaning.
* **Phase 2 (Upcoming):** Automated data ingestion via Python scripting.
* **Phase 3 (Upcoming):** Cloud deployment and live workflow automation using n8n.

## Tech Stack
* **Database:** PostgreSQL (Supabase)
* **Transformation:** Raw SQL 
* **BI & Analytics:** Power BI *(Coming soon)*
* **Automation:** Python & n8n *(Coming soon)*

## Phase 1: Database Setup & SQL Cleaning
In this phase, I established a control sample of live job offers and utilized PostgreSQL to clean, standardize, and transform messy market data into production-ready insights.
