Canadian Tech Jobs Analytics Pipeline

Project Overview

This project analyzes Canadian tech job postings to identify hiring trends, salary patterns, and in-demand skills.

The goal is to build the project step by step while learning real-world data skills, including SQL, PostgreSQL, Python, automation, and data visualization.

The project begins with manual data collection and cleaning and will gradually evolve into an automated analytics pipeline.

---

Objectives

- Collect Canadian tech job postings
- Store job data in PostgreSQL
- Clean and standardize salary information
- Analyze hiring trends using SQL
- Build dashboards in Power BI
- Automate data collection with Python
- Automate workflows using n8n
